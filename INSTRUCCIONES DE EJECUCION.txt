TODO CON DOCKER:
	en la carpeta principal del proyecto hacer:
  docker compose up -d --build

  Espera a que termine la compilacion (la primera vez tarda unos minutos porque compila backend y frontend dentro de Docker).
  La base de datos se crea y carga sola la primera vez.

  Luego con  docker compose up:

  Aplicacion funcional:  http://localhost:8081
  API:         http://localhost:8080
  PostgreSQL:  localhost:5435 (postgres/postgres, BD tareas_db)

  Usuario de prueba: admin / admin123


PARA REINICIAR LA BASE DE DATOS DESDE CERO:
  docker compose down -v   (borra el volumen de datos)
  docker compose up -d --build

CONSULTAS DE LAS 8 PREGUNTAS (opcional, por consola):
  docker exec -i control-2-bda-2026-1-db-1 psql -U postgres -d tareas_db < runStatements.sql
  (verificar el nombre del contenedor con: docker ps)
